Assignment Planner API Documentation

Introduction

This API documentation provides comprehensive details on how to interact with the Assignment Planner API.

Base URL

The base URL serves as the starting point for all API requests:

http://localhost:3001/assignmentPlanner

Create a New Assignment

POST /assignments

Create a new assignment by providing details in the request body:

                {
                    "assignment_name": "Weekly Report 5",
                    "difficulty_level": "Medium",
                    "due_date": "2024-03-06",
                    "instructor_first_name": "Eyhab",
                    "instructor_last_name": "AI-Masri",
                    "status": "Not Done"
                  }

Success response (200 OK):

                {
                "Success": "Successful: Record was added!."
                }

Error response (400 Bad Request):

                {
                "Error": "Failed: Record was not added." 
                }

Retrieve a Specific Assignment by ID

GET /assignments/:assignment_id

Get details of a specific assignment by its ID.

                {
                "Success": "Successfully finished.",
                "data": [
                  {
                    "assignment_id": 1,
                    "assignment_name": "Stack Full Development",
                    "difficulty_level": "Hard",
                    "due_date": "2024-02-23",
                    "instructor_first_name": "Eyhab",
                    "instructor_last_name": "AI-Masri",
                    "status": "Not Done"
                  }
                ]
              }

Success response (200 OK):

Error response (404 Not Found):

                {
                "Error": true,
                "Message": "Assignment not found"
                }

Update an Existing Assignment

PUT /assignments/:assignment_id

Updates an existing assignment. Provide the assignment ID and details to update in the request body.

                {
                    "assignment_name": "Weekly Report 5",
                    "difficulty_level": "Medium",
                    "due_date": "2024-03-06",
                    "instructor_first_name": "Eyhab",
                    "instructor_last_name": "AI-Masri",
                    "status": "Done"
                  }

Success response (200 OK):

                {
                "Success": "Successful: Record was updated!."
                }

Error response (400 Bad Request):

                {
                  "Error": "Failed: All fields are required and must not be empty."
                }

Delete an Assignment

DELETE /assignments/:assignment_id

Deletes a specific assignment by its ID. Also please make sure to provide the assignment ID in the URL.

Success response (200 OK):

                {
                  "Success": "Successful: Record was deleted!"
                }

Error response (404 Not Found):

              {
                "Error": true,
                "Message": "Record not found."
              }
